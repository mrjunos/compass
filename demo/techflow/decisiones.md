# TechFlow Agency — Log de Decisiones

## 2026-03-15 — Cambio de canal de comunicación con clientes
**Decisión:** Migrar comunicación principal con clientes de email a WhatsApp Business.
**Contexto:** Tres clientes se quejaron de tiempos de respuesta por email (promedio 8 horas). WhatsApp tiene respuesta promedio de 45 minutos.
**Impacto:** Todos los proyectos activos migrados. Email solo para documentos formales (propuestas, contratos, facturas).
**Responsable:** Camila

---

## 2026-03-01 — Aumento de precios Q2 2026
**Decisión:** Aumentar precios base en 15% a partir del 1 de abril de 2026.
**Contexto:** Costos operativos subieron (software, freelancers, equipos). Última actualización de precios fue hace 18 meses.
**Excepción:** Clientes con proyectos activos o en negociación antes del 1 de abril mantienen precios anteriores.
**Responsable:** Andrés + Camila

---

## 2026-02-10 — Stack frontend estándar: React + Vite + Tailwind
**Decisión:** Estandarizar todos los proyectos frontend en React + Vite + Tailwind CSS. Descontinuar uso de WordPress para proyectos nuevos.
**Contexto:** WordPress generaba el 60% del tiempo de mantenimiento. React + Vite reduce tiempo de build y mejora performance del cliente.
**Excepción:** Clientes que ya tienen WordPress pueden mantenerlo si no está en scope reemplazarlo.
**Responsable:** Andrés

---

## 2026-01-20 — Política de retainer mensual
**Decisión:** Ofrecer retainer mensual de USD $300 para mantenimiento y optimizaciones. Mínimo 3 meses de compromiso.
**Contexto:** Clientes pedían soporte post-entrega pero no había un modelo claro. El retainer formaliza la relación y genera ingreso recurrente predecible.
**Incluye:** Hasta 4 horas de trabajo, reportes mensuales, monitoreo de uptime.
**Horas adicionales:** USD $80/hora.
**Responsable:** Camila
